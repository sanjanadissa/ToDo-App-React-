📝 ToDo App – React
A simple and responsive ToDo list application built using React and Vite.

Users can:

Add tasks

Mark tasks as completed

Delete tasks

Tasks are stored in the browser using localStorage.

💡 Features
✅ Add new tasks

🔄 Toggle completion

❌ Delete tasks

💾 Saves in localStorage

📱 Responsive UI
